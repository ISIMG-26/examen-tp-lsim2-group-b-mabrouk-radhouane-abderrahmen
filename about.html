A propos de TechZone

Notre mission

TechZone est un mini-projet pedagogique realise dans le cadre du module Technologies & Programmation Web (LSIM 2). L'objectif est de mettre en pratique les technologies du web : HTML5 semantique, CSS3, JavaScript natif, AJAX, PHP et MySQL.

Fonctionnalites

  • Catalogue de produits avec recherche et filtrage en AJAX
  • Inscription et connexion securisees (mot de passe hashe)
  • Panier complet (ajouter, modifier la quantite, supprimer)
  • Validation des formulaires cote client et cote serveur
  • Interface responsive et coherente sur toutes les pages
